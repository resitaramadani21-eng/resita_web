Pengertian Cloud Computing

Cloud computing adalah teknologi yang memungkinkan pengguna mengakses server, penyimpanan, database, dan aplikasi melalui internet tanpa harus memiliki perangkat keras sendiri.

Karakteristik Cloud Computing

  • On-demand self service
  • Broad network access
  • Resource pooling
  • Rapid elasticity
  • Measured service

Model Layanan Cloud

  • IaaS - Infrastruktur sebagai layanan
  • PaaS - Platform untuk pengembangan
  • SaaS - Software siap pakai
